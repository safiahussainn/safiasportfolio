HTML projects: developing a website

In this project I created a website for a client, which allows users to purchase merchanside and create bookings for the band.

Live Link

Project Overview

The software used to successfully create a website for The Sharmanaires, was Notepad++. This was because it supports multiple programming languages, efficient for handling of large text files, and has a user-friendly interface. HTML has also been used for each webpage to allow me to input content like text and images into the webpage. PHP is used on the bookings page, to allow a user to submit their details, which will then be sent to the email address of The Sharmanires. CSS has been used to create the format of the website, including the navigation bar, header, body and footer. This has been applied to every webpage to implement the format, making the website consistent in its layout. JavaScript was used in the footer to create window popups and message boxes when the buttons are pressed. The 4 programming languages used, have helped me to create a user-friendly website, including interactivity and accessibility and increasing the quality of the website.
